# Tetris
Tetris game developed using SFML2 framework.

![Sample](https://github.com/vgomeniuk-games/Tetris/blob/master/assets/animation.gif)

## Development Platform/Tools:
 - Ubuntu 14.04
 - Qt 5.7.0 (64 bit)
 - GCC 4.8.4
